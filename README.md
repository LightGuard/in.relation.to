# Readme

Awestruct powered version of in.relation.to.

To build the website, install [Awestruct](http://awestruct.org).

Tested with:

- ruby 1.9.3-p194
- awestruct 0.4.6

    gem install awestruct --version  0.4.6

Then run

    awestruct -d

and point your browser to <http://localhost:4242>
